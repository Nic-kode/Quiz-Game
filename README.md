# 🧠 Quiz Game

A dynamic quiz game that fetches real-time questions from an external API.

## 🚀 Features
- Fetches questions from an API
- Multiple categories and difficulties
- Score tracking system
- Responsive UI

## 🛠️ Tech Stack
- React
- CSS
- REST API
